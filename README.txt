== Description ==

GLAMI feed generator + PiXel is an extension built for GLAMI, an engine that focuses on all styles of fashion, apparel and accessories.

GLAMI feed generator + PiXel is an essential tool for any online store featured on the Glami product search engine. With GLAMI feed generator + PiXel for WooCommerce
you update your products daily on Glami, automatically creating the XML that contains all the features required by it, as well as integrating the functionality of Glami Pixel.

== Installation ==

1. Upload `glami-feed-generator-pixel-for-woocommerce.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
